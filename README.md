# Bahia-Kino-mites
Supplementary tables (S1-S5) for intertidal soil mite study in Bahía de Kino, Sonora, Mexico

Table S1: Collection dates, locations and plant coverage data for each sample;
Table S2: Soil analysis data;
Table S3: Community matrix (all arthropods);
Table S4: Population densities of major arthropod taxa;
Table S5: Generalized linear regression analyses of count data by transect position and study area
